USEAGE: trec -l [options]
  Show records.

Options:
  Style:
    --simple      : Simple format. (default)
    --full        : Full format with recrd ID.
    --collapse    : Collapses durations by test name.
  Format:
    -c --csv      : CSV
    -j --json     : JSON
    -m --markdown : Markdown table
    -p --pjson    ; Pretty JSON
    -t --text     : Plain text
  Filters:
    --no-filter   : No filtering.
    --today       : Filters start_time by today.
    --all-days    : No filtering by start_time.
    --latest-only : Keeps only the latest record for each test name.
