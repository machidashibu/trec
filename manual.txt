USEAGE: trec [mode] [options]
Mode:
  -r --recording : Start recording with label. (default)
  -l --lookup    : Lookup recorded information by the label or the pattern or date.
  -d --delete    : Delete record by ID.
  -h --help      : Show help

USEAGE: trec label
  Records time duration with the label and the memo.
  Stop recording when input CTRL+C.
Options:
  label : Label of record.

USEAGE: trec -l [options]
  Show records.
Options:
  --format-simple : Simple format. (default)
  --format-full   : Full format with recrd ID.
  --no-filter     : No filtering.
  --today         : Filter by today.
  --all-days      : No filtering by start_time.
  --latest-only   : Keep only the latest record for each unique label.

USEAGE: trec -d id
  Delete record.
Options:
  id : Target record id.
